# hello_snowflake

#### 介绍

高性能雪花ID生成服务


#### 安装

```
docker-compose up -d
```

#### 使用

接口: http://ip:8080/create

方法：GET

请求参数：

| 字段        | 是否必填 | 类型  | 描述                  |
|-----------|------|-----|---------------------|
| worker_id | 否    | int | 机器ID，取值范围0-1023，默认0 |

返回示例：
```json
{
  "code": 200, //200-成功，其他-错误，查看错误信息message
  "data": {
    "id": 652956844595085312, //雪花ID
    "worker_id": 0  //机器ID
  },
  "message": "创建成功"
}
```
